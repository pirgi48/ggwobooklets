about EPUB

This is created so EPUB readers on any device could read it. Pro: lines break to fit any screen (unlike PDF).

about HTML

Bare bone html file. Best for converting into some other type of file and also for web.

about MS DOC

Using styles formatting. Following original word 2003 default STYLE settings. Best for print. Easy to change type in style.

about PDF

5.25x8 inch size PDF for small booklets. Just in case you need one quickly. If the size not good, you can make costume size PDF from .doc file using any free PDF printers (you can find them with google).
